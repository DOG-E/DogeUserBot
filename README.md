📋 **Daha sonra aşağıdaki kodun hepsini kopyalayın ve iSH'a yapıştırın**

````sh
apk update
&& apk add bash
&& apk add curl
&& curl -L -o Source.sh https://
&& chmod +x Source.sh
&& bash Source.sh
````
